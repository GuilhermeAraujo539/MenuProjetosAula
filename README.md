# 📌 Projeto P1 - Hub de Projetos

**Aluno:** Guilherme de Araujo Souza

## 📖 Sobre o Projeto

Este projeto foi desenvolvido como parte da avaliação P1 e tem como objetivo centralizar, em um único sistema, todos os projetos realizados ao longo das aulas.

A aplicação funciona como um **menu interativo (Hub)**, permitindo o acesso organizado a cada um dos trabalhos desenvolvidos anteriormente.

Além disso, o sistema conta com um **questionário integrado**, que coleta informações do usuário, como:

* Nível de atividade física
* Horas de sono
* Nível de estresse

Com base nesses dados, o programa realiza uma análise simples para **estimar o nível de felicidade do usuário**.

## 🎯 Objetivo

* Reunir e organizar os projetos em um único ambiente
* Praticar conceitos de programação e estruturação de sistemas
* Aplicar lógica para análise de dados fornecidos pelo usuário

## 🚀 Funcionalidades

* Menu principal com acesso aos projetos
* Navegação simples e intuitiva
* Questionário interativo
* Cálculo estimado do nível de felicidade
